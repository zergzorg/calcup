## Phase 0: Результаты исследования

### Текущее состояние кодовой базы
- **Стек**: Vue 3.5, TypeScript, Vite, vue-i18n (EN/RU), plain CSS
- **Роутинг**: ручной через `window.location.pathname` (Vue Router отсутствует)
- **Существующие страницы**:
  - `/` — Рабочий стол (виджеты: Pomodoro, TaskPlanner, DateTimer, SoundMachine, SnakeGame)
  - `/credit-calc` — Кредитный калькулятор (аннуитет/дифференцированный, досрочное погашение, печать)

### Анализ конкурентов

| Сайт | Категории | Охват |
|------|-----------|-------|
| calcup.pro | Финансы, Здоровье, Строительство (25!), Алгебра, Конвертеры, Быт | ~50 инструментов |
| calc-up.ru | Учёба/наука, Здоровье, Строительство, Быт, Транспорт, Конвертеры | ~25 инструментов |
| x-calculator.ru | Финансы, Базовые, Дни | ~8 инструментов |
| calculator888.ru | Финансы, Дни/даты, Математика, Генераторы | ~20 инструментов |
| omnicalculator.com | 14 категорий, Физика, Химия, Биология, Еда, Спорт | 3847 инструментов |

**Ключевые выводы**:
- Строительство — самый богатый раздел у ru-конкурентов (calcup.pro: 25 инструментов)
- Дата/время — стабильно популярная категория у всех
- Конвертеры — обязательный раздел (длина, вес, температура, валюта)
- Omni Calculator — мировой эталон: каждый калькулятор это отдельная SEO-страница с объяснением

## Предлагаемая структура сайта

### Главная страница `/`
- Hero-секция: "Все калькуляторы в одном месте"
- Поиск по всем инструментам
- Сетка категорий с иконками (8 разделов + Рабочий стол)
- "Популярные сегодня" — топ-6 инструментов

### Страница категории `/finance`, `/health` и т.д.
- Список всех калькуляторов раздела с описанием
- Карточки с иконками и кратким описанием

### Страница калькулятора `/:category/:tool`
- Сам калькулятор + краткая методология расчёта
- Пример: `/finance/credit`, `/health/bmi`, `/math/percentage`

### Рабочий стол `/workspace`
- Переезд текущей главной (Pomodoro, TaskPlanner, DateTimer, SoundMachine, Snake)
- Ссылка с главной в виде карточки раздела

## ⚠️ Технический роадмап (УСТАРЕЛ)

> Технический план перенесён в `PLAN_architecture.md`. Раздел ниже оставлен только для истории — не использовать как ТЗ.

Актуальный стек: **Vue Router 4 + vite-ssg + PrimeVue 4 (unstyled) + Tailwind CSS 4 + Vitest**.
Реестр (`src/data/calculators.ts`) — единственный источник правды для роутов, sitemap, поиска и хлебных крошек.
