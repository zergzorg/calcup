# Search Keywords Audit

Status: implemented
Created: 2026-04-26
Scope: аудит поисковых ключей для ready-калькуляторов Calcup

## Implementation Notes

Implemented on 2026-04-26:

- `src/composables/useSearch.ts` now normalizes `ё -> е`, `м²/м³ -> м2/м3`, dot separators in units, and compares compact forms for queries like `квтч`.
- Search haystack now includes `description.ru` and `description.en`.
- Added regression tests for `кот`, `котенок`, `пёс`, `щенок`, `платёж`, `счёт`, `м3`, `квтч`, and description-only English query `life stage`.
- Added priority animal aliases and unit aliases to calculator metadata.

## Как Сейчас Работает Поиск

`src/composables/useSearch.ts` ищет только по:

- `title.ru`
- `title.en`
- `tags`
- `aliases`

Поиск не использует `description`, не нормализует `ё -> е`, не приводит слова к начальной форме и работает как `includes()` по lower-case строке. Поэтому важные пользовательские формы нужно явно держать в `tags` или `aliases`.

На момент аудита в каталоге 71 ready-калькулятор. `soon`-карточки в поиск не попадают.

## Критичные Общие Добавления

- Добавлять пары `расчет` / `расчёт`, `счет` / `счёт`, `платеж` / `платёж`, `объем` / `объём`, `отсчет` / `отсчёт`.
- Для животных добавлять бытовые формы: `кот`, `котик`, `котенок`, `котёнок`, `котята`, `щенок`, `щенки`, `пес`, `пёс`.
- Для ремонта добавлять предметные материалы: `профлист`, `профнастил`, `металлочерепица`, `керамогранит`, `ГКЛ`, `минвата`, `пеноплекс`.
- Для единиц добавлять русские сокращения: `кв м`, `м2`, `м3`, `квтч`, `квт ч`, `кВт·ч`, `кмч`.
- Для англоязычного поиска добавлять короткие варианты: `calc`, `converter`, `estimator`, `how much`, где это естественно.

4. Добавить regression-тесты на ключевые запросы: `кот`, `котенок`, `пёс`, `платёж`, `счёт`, `м3`, `квтч`.
